\Large {Le présent rapport synthétise les travaux réalisés dans le cadre de notre système Express IT ; le projet que nous avons réalisés  durant ce stage d'été à l'\textbf{ODC} (ORANGE DIGITAL CENTER)\\\\
}
\Large {L’objectif principal de ce travail consiste à la mise  en place d'un\textbf{ "Système intermédiaire de communication entre les sourds et/ou muets et la société"}\\\\
}

\Large {Ce travail se compose de plusieurs phases de création d’une startup ; commençant par l'analyse du marché, le design et le développement, tout en utilisant des différents outils et en découvrant plusieurs concepts qui nous ont aidés à développer ce système. Ces outils incluent : MERN Stack, le système de gestion de base de données FIREBASE, Google Coud Platform, la configuration du RasberryPi et PYTHON comme langage de développement du modèle d'apprentissage automatique pour le comportement d'IA de notre projet.
\\\\
}
\Large {La suite de ce rapport est organisée comme suit : Le premier chapitre I est consacré à la présentation du projet effectué par le Team Marketing. Le chapitre II traite l'analyse des problèmes et la conception comme mission du Team Développement. Le chapitre III décrit l'utilisation du Machine Learning dans notre projet. Et enfin le dernier chapitre IV présente la mise en œuvre de la solution.
.\\\\
}

\Large {
\textbf{Mots-Clés:} Market - MERN - Machine Learning - Cloud \& IOT 
}
\end{spacing}
